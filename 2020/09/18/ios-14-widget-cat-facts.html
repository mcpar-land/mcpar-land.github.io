iOS 14 Widget - Cat Facts
September 18, 2020

iOS 14’s new homescreen widgets mean only one thing: Ricing.

I wrote a tiny script using Scriptable and one or two public APIs to throw together a widget that shows you a cat fact. And a cat.

let fact_req = new Request('https://catfact.ninja/fact?max_length=125')
let json = await fact_req.loadJSON()
let img_req = new Request('https://cataas.com/cat')
let img = await img_req.loadImage()
console.log(img)
const widget = new ListWidget()
widget.backgroundImage = img
widget.addText(json.fact)

Script.setWidget(widget)
